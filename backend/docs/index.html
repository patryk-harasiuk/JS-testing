Endpoints

  • http://localhost:3000/api/status

    GET

    Responds with server status

  • http://localhost:3000/api/login

    POST

    When correct token is provided within request body it returns user data

  • http://localhost:3000/api/images

    GET

    Returns all images from db

    POST

    Adds new image to db. Requires authorization - correct token must be provided in request body.

  • http://localhost:3000/api/images/:id

    GET

    Returns image of specific id

    DELETE

    Removes selected image from db. Requires authorization - correct token must be provided in request body.
